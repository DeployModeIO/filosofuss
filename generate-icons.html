Generador de iconos PWA — Filosofuss

Dibuja el icono en canvas y descarga todos los PNG necesarios para el manifest y iOS.

Instrucciones

  1. Pulsa «Descargar todos los iconos». El navegador puede pedir permiso para varias descargas: acepta (algunos navegadores las agrupan y debes pulsar «Permitir»).
  2. Guarda los 6 archivos en la carpeta E:\Filosofuss\public\icons\ (créala si no existe).
  3. Los archivos que necesitas son: icon-192.png, icon-256.png, icon-512.png, icon-512-maskable.png, apple-touch-icon.png y favicon.png.
  4. Este paso debe hacerse antes de npm run build, porque Vite copia public/ a dist/ durante el build.
  5. Puedes abrir este archivo haciendo doble clic sobre él: se abre en el navegador sin servidor.
